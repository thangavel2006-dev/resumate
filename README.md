**ResuMate AI** is a professional full-stack web application designed to help job seekers optimize their resumes for **Applicant Tracking Systems (ATS)**. By leveraging AI-driven analysis, it identifies keyword gaps and provides a compatibility score to increase your chances of landing an interview.

## ‚ú® Key Features
* **AI Resume Scanning:** Upload a PDF resume and paste a job description to get an instant match score.
* **Keyword Analysis:** Automatically identifies matched skills and highlights missing keywords recruiters are looking for.
* **User Dashboard:** A personalized space for users to manage their profiles and view history.
* **Scan History:** Track previous analysis results to monitor how your resume improves over time.
* **Secure Auth:** Full registration and login system with local session management.
